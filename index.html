HTML stands for Hyper Text Markup Language

HTML is the standard markup language for creating Web pages

HTML describes the structure of a Web page

HTML consists of a series of elements

HTML elements tell the browser how to display the content

HTML elements label pieces of content

Siin on näide lihtsast HTML struktuurist:

Teooria

ls - näitab mis kasutas sees on

cd "kasuta nimi" - liigub kausta

mkdir "kasuta nimi" - loob kausta

Teekond: cd Documents - cd omanimeline kaust

Koodi kloonimine github'ist: git clone git@github.com....

Terminalis projekti muudatuste üleslaadimine: github'i git add . git commit -m "sõnum,mida sa muutsid" git push

Muudatuste alla tõmbamine github'ist: git pull

<html lang ="et"> määrab dokumendi alguse ja keele, mis aitab otsimootoritel ja ekraanilugejatel sisu mõista.

Päisesse head käib:

<meta charset="utf-8"> – määrab, et tekst on UTF-8 kodeeringus (õiged täpitähed).

<meta name="viewport" content="width=device-width, initial-scale=1"> – seab mobiilse seadme vaate laiuse ja skaala õigeks.

<title> – lehe nimi, mis kuvatakse brauseri sakil.

<link rel="stylesheet" href="assets/css/style.css"> – viide välisele CSS-failile.
