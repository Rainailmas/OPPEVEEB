HTML stands for Hyper Text Markup Language

HTML is the standard markup language for creating Web pages

HTML describes the structure of a Web page

HTML consists of a series of elements

HTML elements tell the browser how to display the content

HTML elements label pieces of content

Koodi algus

<html lang ="et"> määrab dokumendi alguse ja keele, mis aitab otsimootoritel ja ekraanilugejatel sisu mõista.

Päisesse head käib:

<meta charset="utf-8"> – määrab, et tekst on UTF-8 kodeeringus (õiged täpitähed).

<meta name="viewport" content="width=device-width, initial-scale=1"> – seab mobiilse seadme vaate laiuse ja skaala õigeks.

<title> – lehe nimi, mis kuvatakse brauseri sakil.

<link rel="stylesheet" href="assets/css/style.css"> – viide välisele CSS-failile.

HTML Elemendid

<header> – lehe või sektsiooni päis. Tavaliselt sisaldab logo, pealkirja või navigeerimist

<nav> – spetsiaalne ala, kus paiknevad navigeerimislingid. Ekraanilugejad tunnevad selle ära kui navigeerimise bloki.

<section> – temaatiline sisublokk, millel on oma pealkiri

<article> – iseseisev sisutükk, mis võiks ka üksinda avaldatuna tähendust omada

<aside> – lisainfo, nt küljeriba või viited seotud sisule.

<footer> – lehe või sektsiooni jalus (autoriõigused, kontaktid).

padding: – sisemine tühik elemendi sees.

border: – ääris elemendi ümber.

display: – määrab, kuidas element kuvatakse (nt block, inline, flex).

Meedia

img – kõige tavalisem viis pildi lisamiseks. Alt-tekst on kohustuslik ligipääsetavuse jaoks.

picture ja source –võimaldab pakkuda erinevaid failiformaate (nt AVIF, WebP, JPG) ja ekraanisõltuvaid suurusi.

href – lingi sihtkoht (URL).

a – Ümbritsedes see image elemendi ümber muutub pilt klikitavaks.

target="_blank" – Välislinkide puhul kasutada, avab lingi uuel vahelehel.

rel="noopener noreferrer" – Välislinkide puhul kasutada, turvameede, kui link avatakse uuel vahelehel.

loading="lazy" – piltide puhul, laadib pildi alles siis, kui see on vaateväljas.

Video

<iframe> – võimaldab manustada teise lehe sisu (nt YouTube video).

allowfullscreen – lubab videol täisekraanil minna.

video controls – võimaldab manustada video otse lehele.

source – mitme formaadi tugi (WebM, MP4).

track – võimaldab lisada subtiitreid.

SVG

<svg> – vektorgraafika otse HTML-is.

<img> – SVG pildi lisamine.

<object> – interaktiivne SVG lisamine.
